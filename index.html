// Questions

Good to know.

Do I need an OpenAI or Claude account?

You bring your own API key (Claude, GPT, DeepSeek, Mistral) — or run fully free with a local model via Ollama. AG3NT never charges per use; you only pay your provider, if any.

Where does my data go?

Nowhere. Everything runs on your Mac. API keys live in your macOS Keychain. No cloud account, no server, no telemetry of your content.

What's the difference between Basic and Pro?

Basic gives each assistant 40 memories. Pro makes memory unlimited (semantic search) and lets you drop in documents. The Pro memory engine downloads once and runs entirely on-device.

Subscription?

No. One-time purchase, yours forever, with a year of free updates. You already pay your AI provider — we don't double-charge you.
